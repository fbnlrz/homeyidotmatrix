Steuern Sie iDotMatrix BLE-Pixel-Displays von Ihrem Homey Pro aus.
Zeigen Sie mühelos Lauftexte, Uhren, Countdowns und Ihre bevorzugten
Pixel-Art-Animationen an.

Diese App erweckt Ihre Matrix-Displays zum Leben, indem sie diese
direkt in Ihre Homey-Flows integriert. So können Sie Sensordaten,
Benachrichtigungen und vieles mehr visualisieren.
